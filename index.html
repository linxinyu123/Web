Script para ajudar a depurar a função "classify" (2º Projeto) 27 maio 2025, 15:58 • João Pedro Faria Mendonça Barreto Nesta última semana do 2º Projeto, é esperado completarem o trabalho implementando a última função, "classify". Como essa função, nas suas múltiplas etapas, calcula diferentes matrizes, para confirmarem se a vossa implementação está correta, devem confirmar se os valores das matrizes intermédias estão corretos. Para ajudar nessa verificação, hoje publicámos na secção "Projeto" um script python (debug-classify.py) que, para um dado conjunto de ficheiros (m0, m1, imagem de entrada), apresenta no ecrã o conteúdo esperado de todas as matrizes obtidas em cada fase da classificação. Cada grupo deve usar essa informação como referência para conferir se a sua solução está a chegar aos mesmos valores. Notem que, ao contrário do vosso projeto (que recebe ficheiros com valores representados em binário), este script recebe ficheiros em formato de texto/ASCII. Para tal, devem usar as variantes de texto que também foram disponibilizadas em classify-files.zip. Bom trabalho!
